**Free & Open Source Fake Chat/Social Post Generator**

Create realistic mockups for WhatsApp, iMessage, Discord, Slack, ChatGPT, Claude, Instagram, and X (Twitter) - completely free, no watermarks, no limits.

## Features

- **Chat Mockups**: WhatsApp, iMessage, Discord, Slack
- **AI Chat Mockups**: ChatGPT, Claude
- **Social Post Mockups**: Instagram, X (Twitter)
- **Light/Dark Theme**: Toggle between themes
- **PNG Export**: Download mockups as images
- **AI Content Generation**: Auto-generate realistic conversations (coming soon)
- **100% Free**: No watermarks, no limits, no signup required
- **Open Source**: MIT License

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- html-to-image (Export)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/mrsarac/fakely.git
cd fakely

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Docker Deployment

```bash
# Build and run with Docker Compose
docker compose up -d
```
